### DDS Platform

------



DDS Platform은 기구품질(CAD,CAE,Test,측정) 데이터를 기반으로 데이터 생성, 예측모델 개발, 배포를 편리하게 자동화하는 **통합 플랫폼**으로써,



웹 기반의 UI를 통해

- CAE Process 자동화 워크플로우의 작성
- 자동학습을 통한 예측 모델 생성
- 시뮬레이터 배포 자동화

등의 다양한 기능을 제공합니다
