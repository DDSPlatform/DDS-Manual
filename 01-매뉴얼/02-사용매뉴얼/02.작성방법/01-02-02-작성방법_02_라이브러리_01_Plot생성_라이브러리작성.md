#### 1. Plot 생성하는 스크립트 작성 방법

```python
# plot 1장의 시작 부분
plt.figure()

# 실제 Plot 작성하는 부분 (사용자가 직접 작성)

# 로컬과 다르게 이 부분은 작성하지 말 것
#plt.show()

# Plot 1장 종료부분, 'temp.jpg'와 dpi를 임의로 변경하지 말 것
plt.savefig('temp.jpg', dpi=350)
```
